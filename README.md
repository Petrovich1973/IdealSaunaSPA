# IdealSaunaSPA
Landing Page for Ideal Sauna

================================

Установка:
перед сборкой приложения необходимо установить следующие зависимости -
- babel-cli@6.26.0
- node-gyp@3.6.2
- npm@5.3.0
- uglify-js@3.0.22
- webpack@3.5.5
- webpack-dev-server@2.7.1

(посмотреть установленные глобально зависимости можно так - 'npm list -g --depth=0')
